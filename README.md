## ✨ Features

- ➕ Add, ✏️ Edit, 👁️ View, 🗑️ Delete student records
- 🔍 Live search by name or roll number
- 📊 Stats — total, active, avg GPA, departments
- ✔️ Form validation with inline error messages
- 🏷️ Color-coded status badges (Active / Inactive / On Leave)

---

## 🛠️ Tech Stack

- **React 18** with `useState` hooks
- **Vite 5** for fast dev/build
- **Custom CSS** — no UI libraries, fully hand-crafted styles

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/student-management-system.git
cd student-management-system

# Install & run
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

---

## 📖 How to Use

| Action | How |
|---|---|
| Add student | Click **+ Add Student**, fill the form, submit |
| Edit student | Click **Edit** on any row, update fields, save |
| View details | Click **View** on any row |
| Delete student | Click **Delete**, confirm in the prompt |
| Search | Type in the search bar — filters by name or roll no |
